# Amazon Consumer Behavior Analysis

This project explores how age and gender influence purchasing behavior, review engagement, and satisfaction among Amazon customers.

## Objective
The main objective of this analysis is to understand demographic differences in:
- purchasing frequency,
- product category preferences,
- browsing behavior,
- review behavior,
- and customer satisfaction.

## Data
The dataset comes from a survey published on Kaggle and collected via Google Forms.
Responses were gathered primarily from the Indian subcontinent, as well as Indian residents in European and Middle Eastern regions.

## Methodology
- Data cleaning and preprocessing in R
- Handling ordinal and categorical variables
- Creation of dummy variables for multi-response questions
- Descriptive statistics and data visualization
- Correlation analysis
- Linear regression analysis

## Tools
- R
- tidyverse
- ggplot2

## Files
- `Amazon consumer Behaviour.R`: full R script for data cleaning, analysis, and visualization
- `Amazon Consumer Behavior Report.pdf`: written report summarizing methodology and results

## Data availability
The dataset is publicly available on Kaggle.
Due to file size and licensing considerations, it is not included directly in this repository.
